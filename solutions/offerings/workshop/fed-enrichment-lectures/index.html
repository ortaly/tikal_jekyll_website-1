Moving to Client-side Application from Server Page Application

Most Service Oriented Applications and applications that want to scale quickly, provide a rich client interface and provide support for multiple client platforms will eventually neet to seperate the client & server and offload most or all of the view rendering tasks to the client while providing a simple api for comunication with the server.
See more at: http://www.tikalk.com/js/moving-client-side-application-server-page-application#sthash.U8nE2qWu.dpuf

Time Frame : 20 min. 
Lecturer : Assaf Ganon - Tikal's JS group leader. 
Price : F.O.C.

Introduction to Node.js & CoffeeScript

JavaScript is a powerful, general purpose, language that combines Object-Oriented and Functional Programming. A new wave of languages that compile into JavaScript emphasize the power of this combination by suggesting better syntax.

This session will be an overview of Node.js, an event-driven, non-blocking I/O platform that is used for building fast and scalable network applications using JavaScript. A part of the talk will be an overview of CoffeeScript, a language that compiles into JavaScript, and the ways to combine it with Node.js.
 
Time Frame : 50 min. 
Lecturer : Adi Baron - Tikal's Java group member.
Price : F.O.C.

Dive into OOP & Architecture in Javascript

Building javascript applications is easy when using object oriented concepts. In this session we’ll dive into the world of object oriented design and programming through real world examples. we’ll see how to transform any code to a neat, professional object oriented javascript.
